## 📚 Notebooks Overview

### 1️⃣ Pandas — Data Manipulation [`01_pandas.ipynb`](notebooks/01_pandas.ipynb)
> **"The Excel of Python — but 100x more powerful"**

| Topic | Description |
|-------|-------------|
| Series & DataFrame | Core Pandas data structures |
| loc & iloc | Row/column selection methods |
| Importing Data | Reading CSV and JSON files |
| Filtering | Boolean filtering & conditionals |
| Aggregation | `groupby`, `sum`, `mean`, `count` |
| Data Cleaning | Handling nulls, duplicates, type fixing |
| Add / Remove | Columns and rows manipulation |

### 2️⃣ Matplotlib — Data Visualization [`02_matplotlib.ipynb`](notebooks/02_matplotlib.ipynb)
> **"Turn raw numbers into visual stories"**

| Topic | Description |
|-------|-------------|
| Plot Customization | Colors, markers, linestyles |
| Labels & Ticks | Axes labels and tick formatting |
| Bar Charts | Categorical data comparison |
| Pie Charts | Proportional data display |
| Scatter Plots | Relationship between variables |
| Histograms | Distribution visualization |
| Subplots | Multiple plots in one figure |
| Matplotlib + Pandas | Direct integration with DataFrames |

---

### 3️⃣ NumPy — Numerical Computing [`03_numpy.ipynb`](notebooks/03_numpy.ipynb)
> **"The math engine that powers all of Machine Learning"**

| Topic | Description |
|-------|-------------|
| Arrays & Shapes | 1D, 2D, nD array creation |
| Slicing | Array indexing and slicing |
| Arithmetic Operations | Element-wise math |
| Math Functions | `sqrt`, `round`, `floor`, `ceil`, `pi` |
| Broadcasting | Operations on different-shaped arrays |
| Comparison Operators | Boolean arrays & masking |
| Aggregate Functions | `sum`, `mean`, `min`, `max`, `std` |
| Filtering | `np.where` and boolean indexing |
| Random Numbers | Uniform, integer, and random operations |

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/data-science-fundamentals.git
cd data-science-fundamentals
```

### 2. Install dependencies
```bash
pip install pandas numpy matplotlib jupyter
```

### 3. Launch Jupyter
```bash
jupyter notebook
```

Then open any notebook from the `notebooks/` folder.
